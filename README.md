grunt-vue-resource-demo
========================

grunt + vue + vue-resource

安装 CLI 

```
  npm install -g grunt-cli
```

安装Grunt 和 grunt插件

```
  npm install --save-dev grunt
```

```
  npm install  --save-dev grunt-*
```

compare the two commands following 

```
  grunt server
```

or

```
  grunt server:dest
```
